# 关于 Chapter2_ReBuild

实质上和 `Chapter2` 没什么区别，不过为了更加美观的代码风格和更加格式化的修改，对代码进行了重构。现在只需要在 `prompt.py` 文件中添加自定义预设的提示词，
并接入哈希表即可调用，允许使用多种提示词。

## 对文件的说明

- `chat_assisant_AI.py`：用于向服务端发起请求并返回结果，解析返回的json文件并流式输出。
- `main.py`：`main`函数所在位置，作为程序启动端；
- `prompt.py`：储存提示词并提供哈希方式，可以通过修改 `main` 函数的方式实现自定义 or 预设提示词；每次启动程序会重置。
- `config`：将获得的 API Key存入其中，程序会自行读取。
